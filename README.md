# Дневник тренировок в бассейне

Веб-приложение (HTML/CSS/JS) для организации тренировочного процесса в бассейне:
фиксация времени и дистанции по отсечкам, сохранение результатов, графики прогресса
и накопление общего тренировочного объёма.

## Документы
- **`ТЗ.md`** — что строим: требования, экраны, модель данных, формулы, критерии приёмки.
- **`CLAUDE.md`** — как работать: стек, ограничения, соглашения, порядок реализации.

## Как запускать
Приложение статическое, но ES-модули и service worker требуют локального сервера.
Рекомендуется dev-сервер без кэширования — правки видны сразу после reload:

```bash
python3 serve.py          # http://localhost:8145
python3 serve.py 9000     # другой порт
```
Обычный сервер тоже работает, но из-за кэша/PWA правки могут «залипать»:
```bash
python3 -m http.server 8080
```

> Если после изменений видите старую версию — это service-worker-кэш. Проще всего
> открыть на **другом порту** (новый origin без кэша и SW) или один раз очистить
> данные сайта в браузере.
