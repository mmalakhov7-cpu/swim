# Бэкенд на Google Sheets (Apps Script)

Хранит настройки, шаблоны тренировок и результаты в Google-таблице и отдаёт их
приложению по HTTP (JSON). Заменяет/дополняет localStorage.

Таблица: https://docs.google.com/spreadsheets/d/1dBZ_D0bHgAXHHpOg4su7PuUdAtPXHzcxCwcIfX2bRpE/edit

## Деплой (делаешь ты)

1. Открой таблицу → **Расширения → Apps Script**.
2. В файл `Code.gs` вставь целиком содержимое [`Code.gs`](Code.gs) (замени всё). **Сохрани**.
3. Сверху выбери функцию **`setup`** → **Выполнить**. Разреши доступ к таблице.
   Появятся листы: `Settings`, `Templates`, `TemplateTasks`, `Sessions`, `Results`.
4. **Развернуть → Новое развёртывание** → шестерёнка → **Веб-приложение**:
   - Выполнять как: **Я**
   - Есть доступ: **Все (Anyone)**
   - **Развернуть** → разреши → скопируй **URL веб-приложения** (заканчивается на `/exec`).
5. **Пришли мне этот URL** — подключу приложение.

Проверка: открой URL в браузере — должен вернуться `{"ok":true,"data":{...}}`.

> После правок в коде: **Развернуть → Управление развёртываниями → ✎ →
> Версия: Новая → Развернуть**. URL остаётся прежним.

## Листы (что где лежит)

| Лист | Содержимое |
|---|---|
| **Settings** | `key`/`value`: poolLength, keepAwake, showTenths |
| **Templates** | одна строка на шаблон: id, название, число заданий, общий метраж |
| **TemplateTasks** | по строке на задание: шаблон, порядок, название, дистанция, отдых, **описание** (удары/дыхание/лопатки) |
| **Sessions** | по строке на проведённую тренировку: дата, время, объём, время под нагрузкой, **плотность %** |
| **Results** | по строке на задание в тренировке: время, проплыто, сплиты (в колонке `splitsJSON`) |

Шаблоны в `Templates`/`TemplateTasks` можно **редактировать прямо в таблице** —
приложение подтянет изменения.

## API (для фронта)

Один эндпойт — URL веб-приложения.

- **GET** `…/exec` → `{ ok, data: { settings, templates, sessions } }` (всё разом).
- **POST** `…/exec` с телом (обычный текст, JSON-строка):
  ```json
  { "action": "saveTemplate", "payload": { … } }
  ```
  Действия: `getAll`, `saveSettings`, `saveTemplate`, `deleteTemplate`,
  `addSession`, `deleteSession`, `importAll`. Ответ: `{ ok: true }` или `{ ok:false, error }`.

> POST шлём **без** заголовка `Content-Type: application/json` (тело — простой текст),
> иначе браузер делает preflight, который Apps Script не обрабатывает (CORS).

## Безопасность

Доступ «Все» = любой, кто знает URL, может читать/писать твои тренировки.
Секретов там нет (данные тренировок), URL — полусекретный. Если захочешь строже —
можно добавить токен в запрос; скажи, добавлю проверку в скрипт.
